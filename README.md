# accountingapp

작성한 코드 3개를 놓고 비교하여, 본인의 의견을 작성하는 포트폴리오
글속에 반드시 들어가야 하는 개념
1)클래스 - 1,2,3단계 모두 포함
2)생성자 - 3단계에만 사용
3)인스턴스 - 2,3단계에서 사용
4)매개변수 - 3단계에서만
5)this - 3단계에서만

세 코드 모두 클래스를 포함하고 있다. <br> 첫 번째 코드에서는 accounting 클래스가 정의되어 있고, <br> 두 번째 코드에서는 account1 클래스가 정의되어 있으며, <br> 세 번째 코드에서는 account2 클래스가 정의되어 있다. <br>
